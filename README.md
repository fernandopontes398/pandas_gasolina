# Análise de Preços da Gasolina no Brasil (2004–2021)

##  Descrição do Projeto
Este projeto tem como objetivo analisar a evolução dos preços da gasolina comum no Brasil ao longo dos anos, utilizando dados públicos. A análise busca identificar padrões temporais, diferenças regionais e comportamentos atípicos nos preços de revenda.
O projeto foi desenvolvido com foco em **análise exploratória de dados (EDA)**, utilizando Python e Pandas, simulando um cenário real de análise para tomada de decisão.

## Fonte dos Dados
Os dados utilizados neste projeto foram obtidos a partir de um conjunto de dados público amplamente utilizado em estudos e exercícios de análise de dados sobre preços de combustíveis no Brasil.
O dataset contém informações agregadas de preços médios de revenda por estado, região e período, sendo utilizado exclusivamente para fins educacionais e de portfólio.
Após o carregamento, os conjuntos foram tratados e unificados em um único DataFrame para facilitar as análises.

## Tecnologias Utilizadas
- Python  
- Pandas   
- Jupyter Notebook (Google Colab)

##  Etapas da Análise
- Carregamento e unificação dos datasets
- Conversão de colunas de data para o formato datetime
- Limpeza e padronização dos dados
- Cálculo de médias, variações percentuais e comparações históricas
- Análises temporais (mensais e anuais)
- Comparações regionais e estaduais
- Identificação de meses com comportamento atípico de preços

## Perguntas Respondidas
Algumas das perguntas exploradas no projeto:

- Qual foi o preço médio da gasolina em determinados meses e anos?
- Esse valor estava acima ou abaixo da média anual?
- O comportamento de um mês específico foi atípico em relação ao histórico?
- Como o preço médio de São Paulo se compara ao Sudeste?
- Existem diferenças significativas entre estados da mesma região?
- Como a variação percentual dos preços evoluiu ao longo dos anos?

## Principais Insights
- Os preços da gasolina apresentam variações consistentes ao longo dos anos, com tendências de alta mais acentuadas do que as quedas.
- São Paulo frequentemente apresenta preços abaixo da média da região Sudeste, indicando maior competitividade.
- Diferenças de poucos centavos entre estados representam variações percentuais relevantes no contexto histórico.
- Alguns meses, apesar de variações pontuais, não podem ser considerados atípicos quando analisados dentro de um histórico mais amplo.

## Limitações
- A análise é baseada em preços médios, não considerando variações individuais por posto.
- Não foram considerados fatores externos como política de preços, inflação ou eventos econômicos.

## Conclusão
O projeto demonstra como dados públicos podem ser utilizados para extrair insights relevantes sobre o comportamento de preços no Brasil. A abordagem aplicada gerou exelentes aprendizados sobre o uso da ferramenta pandas e analises baseadas nos resultados.
